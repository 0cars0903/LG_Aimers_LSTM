## ⚙️ 핵심 방법론

### 1. Custom Loss 직접 설계

상위 20% 고매출 브랜드의 예측 오류가 순위에 결정적 영향을 미친다는 것을 파악하고,
**브랜드 매출 비중을 가중치로 사용하는 Custom MSE Loss**를 PyTorch로 직접 구현했습니다.

```python
# 핵심: 고매출 브랜드일수록 높은 가중치 부여
def custom_weighted_loss(pred, target, brand_weight):
    mse = (pred - target) ** 2
    return (mse * brand_weight).mean()
```

### 2. 다변량 LSTM + 파생변수 6종
